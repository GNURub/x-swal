x-swal

A beautiful replacement for Javascript's "Alert"

So... What does it do?

Hereâ€™s a comparison of a standard error message. The first one uses the built-in alert-function, while the second is using x-swal.

Normal alert

Code:
alert("Oops... Something went wrong!");

Sweet Alert Polymer

Oops... Something went wrong!

    <x-swal type="error">

Pretty cool huh? SweetAlert automatically centers itself on the page and looks great no matter if you're using a desktop computer, mobile or tablet. It's even highly customizeable, as you can see below!

More examples

In these examples, we're using the shorthand function swal to call x-swal.

  • A basic message

          <x-swal>
          <title>Here's a message!</title>
      </x-swal>
    Here's a message!

  • A title with a text under

          <x-swal>
          <title>Here's a message!</title>
          <text>It's pretty, isn't it?</text>
      </x-swal>
    Here's a message! It's pretty, isn't it?

  • A success message!

          <x-swal type="success">
          <title>Something went wrong!</title>
          <text>Oops...</text>
      </x-swal>
    Something went wrong! Oops...

  • A warning message, with a function attached to the "Confirm"-button...

    Are you sure? Your will not be able to recover this imaginary file! Yes, delete it! 
          <x-swal type="warning" showCancelButton="true" closeOnConfirm="false">
          <title>Are you sure?</title>
          <text>Your will not be able to recover   this imaginary file!</text>
          <confirm color="#DD6B55">Yes, delete it!
        </confirm>
      </x-swal>

  • ... and by passing a parameter, you can execute something else for "Cancel".

    Are you sure? Your will not be able to recover this imaginary file! Yes, delete it! No, cancel plx! 
        <x-swal type="warning" showCancelButton="true" closeOnConfirm="false" closeOnCancel="false">
        <title>Are you sure?</title>
        <text>Your will not be able to recover   this imaginary file!</text>
        <confirm color="#DD6B55">
          Yes, delete it!
        </confirm>
        <cancel>No, cancel plx!</cancel>
    </x-swal>

    <script>
        var alert = document.querySelector('x-swal');
        alert.addEventListener('confirmation',         function (isConfirm) {
            if(isConfirm){
              alert.title = "Deleted!";
              alert.text = "Your imaginary file has been deleted.";
              alert.type = "success";
            }else{
              alert.title = "Cancelled!";
              alert.text = "Your imaginary file is safe :)";
              alert.type = "error";
            }
          alert.active();
        });
    </script>

  • A message with a custom icon

    Something went wrong! Oops... 
          <x-swal type="success">
          <title>Something went wrong!</title>
          <text>Oops...</text>
          <img src="images/thumbs-up.jpg"/>
      </x-swal>

Download & install

Method 1: Install through bower:

$ bower install x-swall

Method 2: Download the x-swalCSS and JavaScript files.

Download files

  1. Initialize the plugin by referencing the necessary files:

    <link rel="import" href="../x-swal/x-swal.html">

  2. Call the x-swal after the page has loaded

            <x-swal type="error">
            <title>Error!</title>
            <text>Here's my error message!</text>
          <confirm>
            Yes, delete it!
          </confirm>
        </x-swal>

Configuration

Here are the content that you can use if you pass an element into x-swall element:

Element 		Default value
Description
title null (required) The title of the modal. It can either be added to the object under the key "title" or passed as the first parameter of the function.
text null A description for the modal. It can either be added to the object under the key "text" or passed as the second parameter of the function.
confirm "OK" Use this to change the text on the "Confirm"-button. If showCancelButton is set as true, the confirm button will automatically show "Confirm" instead of "OK".
cancel "Cancel" Use this to change the text on the "Cancel"-button.
img null Add a customized icon for the modal. Should contain the src with the path to the image.

Here are the attributes that you can use into x-swall element:

Attributes 		Default value
Description
type null The type of the modal. x-swal comes with 4 built-in types which will show a corresponding icon animation: "warning", "error", "success" and "info". It can either be put in the array under the key "type" or passed as the third parameter of the function.
allowOutsideClick false If set to true, the user can dismiss the modal by clicking outside it.
showCancelButton false If set to true, a "Cancel"-button will be shown, which the user can click on to dismiss the modal.
closeOnConfirm true Set to false if you want the modal to stay open even if the user presses the "Confirm"-button. This is especially useful if the function attached to the "Confirm"-button is another x-swal.

Methods of the x-swal element:

Methods
Description
active This method active the window alert.
